About ITA-Computations

ITA-Computations is a Powerful cross-platform geospatial app designed for land surveying and engineering professionals as well as students. It provides advanced computational tools for coordinate processing. Whether you're computing bearings, distances, areas, or generating detailed sheets, ITA-Computations has you covered.

Geospatial Tutorials

1. Importing SUR Plugin Into AutoCAD

Watch: SUR Plugin Installation Walkthrough

  1. Open the folder where AutoCAD is installed and browse to the support folder, example C:\Program Files\Autodesk\AutoCAD 2018\Support
  2. Copy and Paste all the files in the SUR OR JOECAD folder into the support folder and close it
  3. Open AutoCAD as an Administrator
  4. Type Menuload in AutoCAD and press Enter
  5. Click Browse to navigate to the Support Folder and open it
  6. From the files of type Menu, Select Legacy Menu Files (*.MNS*.MNU) to show the Menu files. Select JoeCad and Click open and close the menuload dialogue
  7. Go to Customize Quick Access Toolbar Arrow and Select Show Menu Bar to show all the menus including the SUR Menu

2. Creating CSV Files in Excel

Watch: CSV Creation Tutorial

  1. Open Excel and create columns: Northings, Eastings, Label
  2. Enter your survey data (e.g., 34567.891, 12345.678, 'Boundary Corner')
  3. Save as CSV (Comma Delimited) format
# Example CSV Format
Northing,Easting,Description
34567.891,12345.678,Boundary Corner
34612.345,12378.901,Tree Location

3. Importing CSV into AutoCAD Using SUR Plugin

Watch: CSV Import Tutorial

  1. In AutoCAD, Go to SUR Menu
  2. Select Point Plotting
  3. Enter password and click Ok
  4. Select CSV file from the directory and click Open
  5. Enter Absolute Point size, Text Size and Offset Distance of Text, for example 12 Inches or 0.3m respectively
  6. Choose File Format. Choose according to the format of your CSV file (Northings, Eastings, Label) N,E,L and click Ok to view the points in AutoCAD.
  7. Zoom In and Zoom Out using the Scroll of the mouse to connect the points with a line command.

4. Creating Bearing and Distance for your site plan in AutoCAD using Sur Plugin

Watch: Bearing/Distance Tutorial

  1. In AutoCad Select line command and connect all your points in a clockwise direction according to your traverse
  2. In AutoCAD, Go to SUR Menu and Select Bearing And Distance(ft) or (m)
  3. Type LR for Left to Right and press Enter
  4. Enter Text Height and press Enter
  5. Reverse the Bearing Direction (Yes or No): type N for No and Enter
  6. Select all the lines for the Bearing and Distance and press enter to generate the bearings and distances on the lines selected

5. Creating Grid Lines and Grid Text for your site plan in AutoCAD using Sur Plugin

Watch: Grid Lines Tutorial

  1. In AutoCAD, Go to SUR Menu and Select Grid Plotting
  2. Select lower left Corner about 1000 feet to South-West of the plot
  3. Select Upper Right Corner about 1000 feet to North East of the plot
  4. Select Grid lines and enter 1000 for Grid Spacing and click Ok
  5. Enter label Text height for example 20. zero for decimal places and select any color forText Color and click Ok to generate Grid lines and Grid Text

What does this app do?

This app helps you perform various geospatial calculations, including computing bearing, distance, and area from coordinates.

How do I compute bearing and distance?

Enter the coordinates for Point 1 and Point 2, then click "Save and Calculate" to get results.

Can I compute area from coordinates?

Yes, enter the boundary coordinates and click "Save and Calculate" to compute the area.

How do I switch between light and dark mode?

Open the menu and select "Toggle Dark Mode" to switch themes.

How do I download the app?

Select "Download Android Version", "Download iOS Version", or "Download Windows Version" from the menu.

Can I import coordinates from a CSV file?

Yes, click "Import CSV (N, E, L)" and ensure your file includes Northing (N), Easting (E), and Labels (L).

How do I clear all entered coordinates?

Click "Clear All Rows" to remove all input data.

How do I save my calculations?

Click "Save and Calculate" to generate computation sheets.

Can I share the generated sheets?

Yes, you can save them as PDFs or print directly from the app.
